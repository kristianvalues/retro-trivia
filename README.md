# Retro Trivia Game

A terminal-based Python trivia game that tests your knowledge on awesome 90s nostalgia, vintage video games, classic snacks, and toys. 

## 🚀 How to Run
1. Make sure you have Python installed on your computer.
2. Clone or download this repository.
3. Open your terminal, navigate to this folder, and run:
   ```bash
   python retro_trivia.py
